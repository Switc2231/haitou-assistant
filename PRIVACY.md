# 隐私说明

## 简要结论

本项目不提供云服务，也没有作者自建的数据中转、遥测、广告或统计服务。项目作者不会主动收集、上传、出售用户的 API Key、简历、图片、岗位记录或聊天内容。

“不提供云服务”不代表所有功能都完全离线：只有当用户主动启用 DeepSeek AI 时，简历文字和岗位信息才会直接发送给 DeepSeek；只有当用户确认投递时，简历图片和通过安全检查的自我介绍才会通过 BOSS 页面发送给对应招聘方。相关第三方的数据处理受其自身条款与隐私政策约束。

## 本地保存的数据

扩展使用 `chrome.storage.local` 在浏览器本地保存：

- 用户填写的岗位关键词、城市、筛选词和自我介绍；
- 简历文字与简历图片；
- DeepSeek API Key 和 AI 开关；
- 岗位池、投递状态和去重记录。

这些内容不会发送到项目作者自建的服务器；本项目没有遥测、广告或统计代码。浏览器扩展本地存储并不等同于专用密码保险箱，请使用独立 API Key 并自行控制额度。

## 数据何时会离开本机

1. 用户启用 AI 增强后，简历文字和岗位信息会发送到 DeepSeek API，用于岗位判断和生成自我介绍。
2. 用户确认投递后，简历图片和通过安全校验的自我介绍会通过 BOSS 页面发送给对应招聘方。

不开启 AI 时，不会向 DeepSeek 发送简历文字或岗位信息。

## 用户控制

- AI 增强默认关闭。
- 无 AI 自我介绍为空时，扩展不会自行补充默认文字。
- 用户可以在侧栏修改配置、重置岗位池和投递记录。
- 如需彻底清除扩展本地配置和简历内容，可在浏览器扩展管理页移除扩展，或清除该扩展的站点/扩展数据。

## 平台说明

扩展运行在 BOSS 直聘页面，其页面行为和数据处理仍受 BOSS 直聘自身条款与隐私政策约束。使用者应自行确认使用方式符合当地法律、平台规则和账号要求。

本项目仅用于学习研究和个人本地辅助。禁止将其用于绕过风控、规避验证码、批量骚扰、刷屏、多账号自动化、批量采集平台数据、商用代投或其他违反第三方规则的用途。
