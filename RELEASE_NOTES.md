# v0.4.7 发布说明

## 主要变化

- 移除固定职业优先词、排除词和自动兜底招呼语。
- 岗位关键词、筛选词和无 AI 自我介绍均由用户自行填写。
- 加入后台与聊天页双重外发文字安全校验。
- 使用 V3 岗位会话令牌和 V6 图片发送锁。
- 上传前检查当前会话是否已经存在自己发送的图片。
- 图片已发送但文字失败时标记为待人工确认，避免重复上传。
- 公开发布包移除未被 manifest 加载的旧根目录脚本和职业画像文件。
- 新用户城市默认留空，留空时按全国搜索。

## 已知限制

- 仅支持 BOSS 直聘网页版。
- AI 设置界面当前只直接支持 DeepSeek。
- 页面 DOM 改版、验证码、登录状态和平台风控可能中断流程。
- 自动化测试不能替代真实页面的单岗位人工验证。
