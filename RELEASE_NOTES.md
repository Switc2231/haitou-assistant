# 下载前请阅读

该版本仅供学习和个人本地辅助使用。请勿用于批量骚扰、刷屏、绕过平台风控、规避验证码、采集平台数据、多账号自动化或商用代投服务。

本项目不提供云服务，项目作者不通过自建服务器收集或上传用户数据。配置默认保存在用户本地浏览器中；启用 DeepSeek 或确认投递时，相关数据会按功能需要直接发送给对应第三方。

使用本插件前，请自行确认是否符合相关平台规则。因不当使用导致的账号限制、封禁、投诉或其他后果，由使用者自行承担。

如遇验证码、405 页面、账号风险提示或页面异常，请立即停止使用。

# v0.4.7 发布说明

## 主要变化

- 移除固定职业优先词、排除词和自动兜底招呼语。
- 岗位关键词、筛选词和无 AI 自我介绍均由用户自行填写。
- 加入后台与聊天页双重外发文字安全校验。
- 使用 V3 岗位会话令牌和 V6 图片发送锁。
- 上传前检查当前会话是否已经存在自己发送的图片。
- 图片已发送但文字失败时标记为待人工确认，避免重复上传。
- 公开发布包移除未被 manifest 加载的旧根目录脚本和职业画像文件。
- 新用户城市默认留空，留空时按全国搜索。

## 已知限制

- 仅支持 BOSS 直聘网页版。
- AI 设置界面当前只直接支持 DeepSeek。
- 页面 DOM 改版、验证码、登录状态和平台风控可能中断流程。
- 自动化测试不能替代真实页面的单岗位人工验证。
